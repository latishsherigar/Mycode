Hide It version 1.0

 

Retrieving the Data:

1.Open Option:

.Choose the Open option from the menu.

 

2. Open Dialog box:

The following dialog box appears .Select the file which contains your hidden data.

 

 

3.Open Dialog box with Invisible Mode option:

If your Data is hidden in Invisible Mode then click the Invisible Mode checkbox .When the checkbox is clicked the dialog box looks like the screenshot below. Enter the password and then click the OK button.

 

4.Program Showing Hidden Data:

The program then shows the hidden data in the file. The files shown on your screen may  be different than shown here depending upon you data.

 

5. Extract Dialog Box:

Click the Show button on the Toolbar .The following dialog box appears on the screen. Click the Browse button and then choose the Folder where you want to extract your data to.

 

The Folder dialog box looks like the screenshot below.

 

6.Removing Hidden Data:

If you want to remove the hidden data from the main file after extracting the data then click the checkbox given.

 

7.Finished:

Congrats You have successfully retrieved your Data.

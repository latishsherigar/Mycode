SPLITTER:
The splitter can split files of any practical size and split thenm in to user defined size.

FileName: The filename to be splitted.
Outputdir: The output directory in which the splitte dfiles has to be placed.
Type: The memory format in which the files have to be splitted.
Size : The size in which the files have to be splitted.

MERGER :
The merger can merge or join the splitted files snd produce a single file.

Filename :The filename of one of the splitted file which has to be joined.
Outputdir:The output directory in which the splitte dfiles has to be placed.

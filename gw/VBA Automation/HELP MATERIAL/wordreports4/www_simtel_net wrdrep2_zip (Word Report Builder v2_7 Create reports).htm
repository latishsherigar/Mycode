Description:  Word Report Builder v2.7: Create reports

Word Report Builder v2.7 is an easy and convenient tool for the creation
and customization of reports which takes advantage of the formatting and
presentation capabilities of Microsoft Word. The program allows the
rapid construction of reports as forms, columnar reports and master
detail reports. To link to your database, the program dialogue boxes
can be used, or the user may choose SQL.

A GUI style design environment, which allows the user to design their
reports visually inside MS Word. Reports can be saved and then viewed
and printed as pure Word documents - there is no need to send any
additional executables to third parties to whom you wish to send
reports. Word Report Builder allows the creation of Master/Detail style
reports - e.g. Customer/Orders, etc. The power of SQL can be harnessed
for maximum efficiency in the creation of your reports. It is possible
to call the Word Report Builder from other programs.

The combination of the above qualities make the Word Report Builder a
tool which should be considered seriously by any organization with a
need to effectively and efficiently disseminate information both between
their own staff and to clients, suppliers, etc.
